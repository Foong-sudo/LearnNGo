## Description
To develop a system that will improve the quality of education and make learning more engaging towards the demographic of high school and university students.

## SDLC Work Summary
The first step was to choose a sustainable development goal (SDG), and the SDG chosen was quality education (SDG 4). The concept chosen was to create a game-like app which we named LearnNGo. The features and functionalities of the app were simulated using use-case and class diagrams, whilst the UI design concepts were created using Figma and the testing was carried out through the prototyping feature of Figma. The end results were documented through sprint notes which were later put into the report.
